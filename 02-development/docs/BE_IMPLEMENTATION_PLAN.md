## Стек приложения
- **Язык и рантайм:** Python 3.13+
- **API:** RESTful JSON API, эндпоинты реализованы согласно требованиям из `01-analysis/`
- **HTTP:** Flask + flask-cors
- **БД:** SQLite 3 с WAL-режимом, транзакции для бронирований
- **Аутентификация:** phone/OTP flow, Bearer token, dev-реализация выводит OTP в консоль
- **Тесты:** ручное тестирование через браузер и curl

## Правила
- **Атомарность:** операции создания и отмены брони выполняются в транзакциях с блокировкой слота.
- **Идемпотентность:** создание брони требует заголовок Idempotency-Key, повтор с тем же ключом не создаёт дубль.
- **Правило 2 часов:** отмена за ≥ 2 ч до начала – cancelled, места возвращаются; < 2 ч – late_cancel, места не возвращаются.
- **Раздельный учёт:** места и прокатные наборы учитываются независимо.
