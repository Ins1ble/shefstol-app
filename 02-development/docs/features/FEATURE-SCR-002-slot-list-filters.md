# Фича: Список классов и фильтры (SCR‑002, BS‑001)

## 1. Цель
Отобразить каталог предстоящих кулинарных классов с возможностью фильтрации по дате, типу программы, шефу и наличию свободных мест. Обеспечить быстрый переход к карточке класса для записи.

## 2. Требования (источники)
- **Функциональные:** FR‑9 (список классов), FR‑38 (фильтрация)
- **Логики:** LOGIC‑005 (фильтрация и сортировка), LOGIC‑008 (состояния экрана)
- **User Stories:** US‑2, US‑3
- **Use Case:** UC‑3 (просмотр и фильтрация)
- **Экранные требования:** SCR‑002 (`SCR-002-slot-list.md`), BS‑001 (`BS-001-filters.md`)

## 3. Реализация
### 3.1. Файлы
- `frontend/js/slots.js` – функции `loadSlots`, `renderSlots`, фильтры, `initSlotsHeader`
- `frontend/js/api.js` – вызовы `GET /api/slots`, `GET /api/chefs`
- `frontend/js/utils.js` – скелетоны, форматтеры

### 3.2. Список классов (SCR‑002)
- При открытии вкладки «Классы» или после входа вызывается `loadSlots()`.
- Запрос `GET /api/slots` с параметрами фильтров (если применены).
- Состояния экрана:
  - **Loading** – скелетоны карточек (3 шт.).
  - **Content** – список карточек, сгруппированных по дням (sticky‑заголовки).
  - **Empty** – заглушка «Пока нет доступных классов» или «Ничего не найдено по фильтрам» с кнопкой «Изменить фильтры».
  - **Error** – заглушка с кнопкой «Обновить».
- Карточка класса: время, программа, тип (бейдж), шеф, цена, свободно мест (X из Y). Если мест нет – бейдж «Мест нет», карточка некликабельна.
- Pull‑to‑refresh: жест «потянуть вниз» обновляет список (индикатор сверху, контент не сбрасывается в скелетон).

### 3.3. Фильтры (BS‑001)
- Кнопка «Фильтры» в хедере с индикатором количества активных групп.
- Модальное окно (Bootstrap Modal) с полями:
  - **Дата:** пресеты «Сегодня», «Эта неделя», «Выходные» + произвольный диапазон (date_from/date_to).
  - **Тип программы:** чекбоксы «Новичковая» / «Опытная» (OR внутри группы).
  - **Только со свободными местами:** toggle (only_available).
  - **Шеф:** чекбоксы из справочника (загружается через `GET /api/chefs`), мультивыбор (OR).
- Кнопка «Применить» – закрывает модалку, обновляет `loadSlots` с новыми параметрами, обновляет индикатор.
- Кнопка «Сбросить» – возвращает все фильтры к дефолтным значениям (без применения).
- Справочник шефов загружается при открытии фильтров; при ошибке – inline‑сообщение в секции, остальные фильтры доступны.

### 3.4. Сортировка и группировка
- По умолчанию сортировка по `start_at` ASC.
- Группировка по дням: дата выводится как sticky‑заголовок (текстовый, не скрывается при скролле в рамках веб‑реализации просто выводится над группой).
