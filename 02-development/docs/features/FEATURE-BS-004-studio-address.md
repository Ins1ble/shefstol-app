# Фича: Адрес студии (BS‑004)

## 1. Цель
Предоставить клиенту возможность увидеть полный адрес кулинарной студии и, при наличии координат, интерактивную карту с пином. Позволить быстро построить маршрут или открыть адрес во внешнем картографическом приложении.

## 2. Требования (источники)
- **Функциональные:** FR‑9a (адрес студии в карточке класса и деталях брони)
- **Логики:** LOGIC‑006 (отображение адреса с fallback на текст)
- **Нефункциональные:** NFR‑26 (текстовый fallback при недоступности карты)
- **User Stories:** US‑4 (клиент видит все детали класса, включая адрес)
- **Экранные требования:** BS‑004 (`BS-004-studio-address.md`)

## 3. Реализация
### 3.1. Файлы
- `frontend/js/booking.js` – функция `showStudioAddress(address, lat, lng)`
- `frontend/index.html` – контейнер для модального окна (создаётся динамически)
- `frontend/css/style.css` – стили для модального окна

### 3.2. Отображение адреса в карточке класса и деталях брони
- В `SCR‑003` и `SCR‑006` адрес студии выводится текстовым блоком.
- Если API предоставляет координаты (`studio_lat`, `studio_lng`), дополнительно отображается **статическое превью карты** (через Яндекс.Static Maps) с пином.
- При клике на адрес или превью открывается модальное окно `BS‑004`.

### 3.3. Модальное окно «Адрес студии»
- Заголовок «Адрес студии».
- Полный почтовый адрес крупным текстом (`studio_address`).
- При наличии координат – **интерактивная карта** (статическое изображение с кнопкой «Открыть в Яндекс.Картах»).
- Кнопки действий:
  - **«Проложить маршрут»** – открывает Яндекс.Карты с построением маршрута до координат студии (или по текстовому адресу).
  - **«Открыть в картах»** – открывает координаты/адрес во внешнем картографическом приложении.
- Кнопка «Закрыть» в хедере, свайп вниз или тап по бэкдропу – закрыть модалку.

### 3.4. Обработка ошибок и fallback
- Если координаты отсутствуют – карта не показывается, только текстовый адрес и кнопки действий.
- При ошибке загрузки карты (сеть, недоступность сервиса) – вместо карты отображается сообщение «Не удалось загрузить карту» и кнопка «Обновить», текстовый адрес и кнопки действий доступны.
