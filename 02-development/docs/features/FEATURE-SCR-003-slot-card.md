# Фича: Карточка класса (SCR‑003)

## 1. Цель
Показать клиенту полную информацию о выбранном кулинарном классе, чтобы он мог принять решение о записи. Предоставить кнопку «Записаться» (активна при наличии свободных мест) и ссылку на адрес студии с картой.

## 2. Требования (источники)
- **Функциональные:** FR‑9a (карточка класса)
- **Логики:** LOGIC‑002 (доступность), LOGIC‑006 (адрес студии), LOGIC‑008 (состояния экрана)
- **User Story:** US‑4
- **Экранные требования:** SCR‑003 (`SCR-003-slot-card.md`)

## 3. Реализация
### 3.1. Файлы
- `frontend/js/booking.js` – функции `openSlot`, `renderSlotCard`
- `frontend/js/api.js` – вызов `GET /api/slots/{slotId}`
- `frontend/js/utils.js` – форматтеры дат

### 3.2. Отображение
- Открывается при клике на карточку в списке классов (SCR‑002).
- Запрос `GET /api/slots/{slotId}`. В состоянии Loading – скелетон (или спиннер).
- Блоки:
  - Дата и время начала (крупно).
  - Название программы, тип (бейдж), описание (если есть), длительность (если есть).
  - Имя шефа.
  - Адрес студии: текстовый блок, кликабельный (открывает BS‑004). Если есть координаты (`studio_lat`, `studio_lng`) – статическое превью карты Яндекс.Карт с пином.
  - Места: «Свободно X из Y».
  - Прокатные наборы: «Свободно Z».
  - Цена за место, напоминание об офлайн‑оплате.
- Кнопка «Записаться»: активна только если `free_seats > 0`. При отсутствии мест – кнопка disabled с текстом «Мест нет».

## 4. Отправленные промпты (ключевые)
1. «Создай экран карточки класса с блоками программы, шефа, адреса, мест и проката.»
2. «Добавь статическую карту Яндекса при наличии координат.»
3. «Сделай кнопку “Записаться” неактивной при free_seats=0.»

## 5. Ручная проверка (чек-лист)
- [ ] Кликнуть на класс со свободными местами – открывается карточка со всеми полями.
- [ ] Если у класса есть координаты – отображается карта‑превью.
- [ ] Нажать «Записаться» – переход к оформлению (SCR‑004).
- [ ] Кликнуть на класс без мест – кнопка «Мест нет» неактивна.
- [ ] Нажать на адрес – открывается модальное окно BS‑004.
