# 📄 Экспорт дизайн-брифа в PDF

Скрипты для сборки единого PDF-документа из всех файлов дизайн-брифа проекта «Шеф-стол».

## 🎯 Что делает

1. **`build.py`** — собирает все Markdown-файлы из `01-analysis/3-design-brief/` в один `design-brief.html` с:
   - титульной страницей
   - оглавлением с якорными ссылками
   - корректной типографикой и цветовой схемой (янтарный #d97706)
   - разрывами страниц между разделами

2. **`build.sh`** — печатает HTML в PDF через Chrome headless (`design-brief.pdf`).

## 📦 Включённые документы (в порядке оглавления)

1. `design-brief.md` — Обзор брифа

3. `SCR-001-registration.md` — Регистрация / Вход
